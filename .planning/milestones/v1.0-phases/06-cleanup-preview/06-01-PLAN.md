**Step 2: Replace the run_preview stub in commands/clean.rs**

Replace the entire content of `freespace/src/commands/clean.rs` with a full implementation. Keep `run_apply` unchanged (still a stub). The file must contain:

1. **Imports:** `crate::classify::{safety_class, SafetyClass}`, `crate::config::schema::Config`, `bytesize::ByteSize`, `comfy_table::Table`, `serde::Serialize`, `std::path::{Path, PathBuf}`

2. **PreviewEntry struct** (private, derives Debug + Serialize):
   - `path: PathBuf`
   - `total_bytes: u64`
   - `file_count: u64`
   - `safety: SafetyClass`

3. **PreviewResult struct** (private, derives Debug + Serialize):
   - `candidates: Vec<PreviewEntry>`
   - `total_bytes: u64` (sum of ALL candidates)
   - `reclaimable_bytes: u64` (sum of only SafetyClass::Safe candidates)

4. **known_cache_dirs(home: &Path) -> Vec&lt;PathBuf&gt;** (private function, duplicated from caches.rs):
   Return the same 7 paths as caches.rs:
   - `home.join("Library/Caches")`
   - `home.join("Library/Logs")`
   - `home.join("Library/Developer/Xcode/DerivedData")`
   - `home.join(".npm")`
   - `home.join(".cargo/registry")`
   - `home.join("Library/Containers/com.docker.docker")`
   - `home.join(".gradle/caches")`

5. **run_preview(config: &Config, json: bool) -> anyhow::Result&lt;()&gt;** (public):
   - Resolve home via `dirs::home_dir()`, error with `anyhow::anyhow!("Cannot resolve home directory")` if None
   - Get dirs from `known_cache_dirs(&home)`
   - For each dir: skip if `!dir.exists()`, otherwise call `crate::fs_scan::scan_path(&dir, config)` and `safety_class(&dir, &home)`
   - Build `Vec<PreviewEntry>` from results
   - Sort candidates: primary key `safety` ascending (Safe first, via Ord), secondary key `total_bytes` descending (largest first within same safety class). Use `.sort_by(|a, b| a.safety.cmp(&b.safety).then(b.total_bytes.cmp(&a.total_bytes)))`
   - Compute `total_bytes` as sum of all `candidates.iter().map(|e| e.total_bytes)`
   - Compute `reclaimable_bytes` as sum of only `.filter(|e| e.safety == SafetyClass::Safe).map(|e| e.total_bytes)`
   - Build `PreviewResult { candidates, total_bytes, reclaimable_bytes }`
   - If `json`: call `crate::output::write_json(&result)?`
   - Else: call `render_preview_table(&result)`
   - Return `Ok(())`

6. **render_preview_table(result: &PreviewResult)** (private):
   - Create `comfy_table::Table` with header `["Path", "Size", "Files", "Safety"]`
   - For each candidate: add row with `path.to_string_lossy()`, `ByteSize::b(total_bytes).to_string()`, `file_count.to_string()`, `safety.to_string()`
   - `println!("{table}")`
   - `println!("Total: {}", ByteSize::b(result.total_bytes))`
   - `println!("Reclaimable (safe): {}", ByteSize::b(result.reclaimable_bytes))`

7. **run_apply** -- keep exactly as-is (the existing stub with eprintln/write_json).

CRITICAL: run_preview must NOT call `trash::delete`, `std::fs::remove_file`, `std::fs::remove_dir_all`, or any filesystem-modifying function. It is read-only (PREV-02).

CRITICAL: No `println!` or `print!` calls in run_preview itself -- table output goes through render_preview_table, JSON goes through output::write_json. Diagnostic messages use `tracing::warn!` (stderr).
