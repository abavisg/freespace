2. **`pub fn run(path: &Path, config: &Config, json: bool) -> anyhow::Result<()>`:**
   - Validate path exists: `if !path.exists() { anyhow::bail!("path does not exist: {}", path.display()); }`
   - Resolve home: `let home = dirs::home_dir().unwrap_or_else(|| PathBuf::from("/"));`
   - Pre-initialize HashMap with all 14 categories (CRITICAL for CAT-01): `let mut totals: HashMap<Category, (u64, u64)> = HashMap::new(); for cat in Category::all() { totals.insert(*cat, (0u64, 0u64)); }`
   - Walk with `walkdir::WalkDir::new(path).follow_links(false)`:
     - On `Ok(entry)` where `entry.file_type().is_file()`:
       - Skip configured exclusions: `if config.scan.exclude.iter().any(|ex| entry.path().starts_with(ex)) { continue; }`
       - Get metadata, skip on error (log + continue)
       - Use `std::os::unix::fs::MetadataExt` for `metadata.blocks() * 512` as physical size
       - Deduplicate hardlinks: maintain `HashSet<(u64, u64)>` of `(dev, ino)`, skip if already seen
       - Call `classify_path(entry.path(), &home)` to get category
       - Accumulate into totals: `entry.total_bytes += physical; entry.file_count += 1;`
     - On `Err(e)`: log with tracing::warn, continue
   - Build `CategoriesResult` from totals HashMap, sorting categories by `Category::all()` order
   - If `json`: call `crate::output::write_json(&result)?`
   - Else: render table with `comfy_table::Table`. Columns: `Category | Size | Files`. Use `bytesize::ByteSize::b(total_bytes).to_string()` for human-readable size. Add a summary row with totals.
