2. **`pub fn run(path: &Path, config: &Config, json: bool) -> anyhow::Result<()>`:**
   - Validate path: `if !path.exists() { anyhow::bail!("path does not exist: {}", path.display()); }`
   - Collect hidden entries from IMMEDIATE children of `path` only (NOT recursive descent into hidden dirs). Use `std::fs::read_dir(path)` to iterate immediate children:
     - For each `DirEntry`, check `is_hidden(entry.path())` using `classify::is_hidden()`
     - Skip `.` and `..` (read_dir doesn't return them, but be safe)
     - If hidden AND is a directory: call `crate::fs_scan::scan_path(&entry.path(), config)` to get the recursive total size. Create `HiddenEntry { path, size_bytes: scan_result.total_bytes, is_dir: true }`
     - If hidden AND is a file: get metadata, use `std::os::unix::fs::MetadataExt`, size = `metadata.blocks() * 512`. Create `HiddenEntry { path, size_bytes: physical_size, is_dir: false }`
     - On metadata or read_dir errors: log with `tracing::warn!`, skip entry, continue
   - Sort entries by size_bytes descending (largest first)
   - Compute `total_hidden_bytes` = sum of all entry size_bytes
   - Compute `total_hidden_count` = entries.len()
   - Build `HiddenResult { root, entries, total_hidden_bytes, total_hidden_count }`
   - If `json`: `crate::output::write_json(&result)?`
   - Else: render table with `comfy_table::Table`. Columns: `Name | Type | Size`. Type is "dir" or "file". Use `bytesize::ByteSize::b(size_bytes).to_string()` for size. Add a footer row: `Total: {count} hidden items, {total_size}`.

**IMPORTANT (Pitfall 3 avoidance):** Do NOT use WalkDir to recursively list all hidden files. Instead, enumerate hidden items at the top level of the scanned path, then size hidden directories using scan_path. This prevents double-counting (listing .ssh and also listing .ssh/id_rsa).
